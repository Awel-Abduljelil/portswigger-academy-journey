# PortSwigger Lab Tracker

This folder now contains the live progress system for your portfolio.

## Files

- `labs-log.csv` — add one row per completed lab.
- `labs-tracker.md` — generated Markdown table updated automatically by GitHub Actions.

## How to use it

1. Add a new lab row to `labs-log.csv`.
2. Commit and push your changes.
3. The GitHub Action will regenerate `labs-tracker.md` automatically.

This keeps your progress visible, professional, and easy to show to recruiters, clients, and cybersecurity communities.
